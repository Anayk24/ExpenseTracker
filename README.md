## Assignment:ExpenseTracker

A simple Java-based Expense Tracker project that records and manages daily expenses using **Object-Oriented Programming (OOP)** and **file handling**.

## Features
- Add new expenses (category, amount, date)
- Save expenses to a file (`expenses.ser`) using serialization
- View all saved expenses
- Persistent storage (expenses remain saved after program restarts)

## Technologies Used
- Java (OOP, Serialization)
- File Handling (ObjectOutputStream & ObjectInputStream)
